# AI Website Copy Generator for Local Businesses (Task 1)

## 📌 Overview
This project is part of the Prompt Engineering Internship by Future Interns.

The objective is to build a system that generates high-quality, conversion-focused website copy for local businesses using structured AI prompts.

## 🏢 Business Chosen
GlowUp Salon (Hyderabad)

A premium yet affordable salon offering:
- Haircuts & Styling
- Hair Coloring
- Facials
- Bridal Makeup

## 🎯 What This Project Does
This system generates:
- Homepage content
- Service page descriptions
- Call-To-Action (CTA) sections

All content is:
- Simple
- Persuasive
- Website-ready
- Tailored for local business audiences

## 🛠 Tools Used
- ChatGPT (for content generation)
- GitHub (for documentation)
- HTML/CSS frontend (for website display)

## ⚙️ Prompt Workflow

Input → Structured Prompt → AI Output → Refined Content → Website Integration

1. Define business details (input)
2. Use structured prompts (role + format + tone)
3. Generate content using AI
4. Refine output for clarity and realism
5. Apply content to frontend pages

## 🧠 Prompt Strategy
Prompts are designed using:
- Role-based instructions (professional copywriter)
- Structured output sections (Hero, About, Services, CTA)
- Tone control (friendly + premium)
- Conversion-focused messaging

## 📄 Pages Generated
- Homepage
- About Page
- Services Page
- Testimonials Page
- Experience Page

## 📂 Project Structure
- inputs → Business details
- prompts → Prompt templates
- outputs → Generated content
- frontend → Website files

  ## ⚙️ Functional Features Added

To enhance the project beyond static content, the following functionalities were implemented:

### 1. Dynamic Content Generation
- Homepage content updates based on selected tone
- Simulates AI-generated website copy

### 2. Tone Adaptation System
- Premium, Friendly, and Luxury modes
- Updates Hero section, Services, and CTA dynamically

### 3. Service Selection Feature
- Users can select services
- Content updates instantly without page reload

### 4. Booking Form
- Input validation for required fields
- Displays confirmation message

### 5. CTA Optimization
- CTA text dynamically changes based on tone

These features demonstrate how structured prompts can translate into dynamic website behavior.
